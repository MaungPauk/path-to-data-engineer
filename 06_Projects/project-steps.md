## 1. CSV to Database ETL Pipeline

### Overview
This beginner-friendly ETL project teaches how to move data from CSV files into a database.

### Key Steps
- **Extract:** Read data from CSV files using Python or ETL tools  
- **Transform:** Clean data (fix missing values, formats, duplicates)  
- **Load:** Insert data into a database  
- **Schedule (Optional):** Automate with cron or Airflow  

### Skills You’ll Learn
- Data cleaning using Python/SQL  
- Writing transformation rules  
- Loading data into databases  
- Basic automation  

### Tools
- Python (Pandas, SQLAlchemy)  
- MySQL / PostgreSQL  
- Cron / Airflow  
- DBeaver / pgAdmin
